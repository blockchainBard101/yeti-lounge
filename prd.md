**✅ Product Requirements Document (PRD)**  
**Product Name:** Yeti Lounge  
**Version:** 1.0 (Hackathon MVP)  
**Date:** May 2026  
**Submitted for:** Lofi The Yeti CLAY Hackathon Round 2  

### 1. Product Vision & Overview
**Yeti Lounge** is the official decentralized home base for the Lofi The Yeti community on Sui — a chill, fun, and utility-rich **SocialFi platform** that combines social interaction, rewards, events, and transparency.

It serves as the central hub where new and existing Yetis can connect, create memes, learn about Sui, participate in events, and track real-world impact from the **Lofi Foundation** (water/charity projects).  

**Tagline:** “Thaw In. Hang Out. Build Together. 🥶💧”

**Core Promise:** Make onboarding millions feel effortless and fun while creating long-term retention through ownership, rewards, and community-driven culture.

### 2. Objectives & Goals
**Hackathon Goals:**
- Deliver a polished, functional MVP by June 30, 2026.
- Showcase Sui-native features (objects, Kiosk, sponsored txs, SuiNS).
- Win community and judge love through strong Lofi branding and utility.

**Business / Ecosystem Goals:**
- Become the go-to platform for Lofi community engagement beyond X/Telegram.
- Drive $LOFI holder retention and new user onboarding.
- Increase visibility and participation in Lofi Foundation charity initiatives.
- Demonstrate “meme + real building” philosophy.

**Success Metrics (MVP):**
- 500+ active test users.
- 1,000+ on-chain interactions (posts, RSVPs, badge mints).
- Positive feedback from Lofi team/community.
- Smooth UX with <5-second tx confirmations and gasless flows.

### 3. Target Audience
- **Primary:** Current $LOFI and Mystic Yeti NFT holders.
- **Secondary:** New Sui users, meme enthusiasts, builders, and people interested in charity/impact.
- **Persona Example:** “Chill Yeti” — 18-35, crypto-curious, loves memes and lofi beats, wants easy entry into Sui.

### 4. Core Features (MVP Scope)

#### **Onboarding & Profiles**
- Wallet connect with **zkLogin** option for easy entry.
- **Sponsored transactions** for gasless first actions (profile creation, first post).
- Create on-chain **Yeti Profile** (Sui Object) with SuiNS handle (e.g., bard.yeti).
- Visual Yeti avatar customizer (basic accessories via Kiosk).
- Holder verification badges ($LOFI staked / NFT ownership).

#### **Meme Feed & Social**
- Post memes, images, short text (on-chain or hybrid with Walrus for media).
- Reactions, comments, and community voting.
- Reward system: Top voted posts earn small $LOFI drips or badge NFTs (Move contract logic).
- “YEEERRRR” button for fun interactions.

#### **Flurry Dashboard**
- Live community metrics: active users, total posts, $LOFI activity.
- Lofi Foundation impact tracker (water wells donated, liters cleaned — pull public data).
- Leaderboards: Top meme creators, top learners, top donors.

#### **Events & Perks**
- On-chain event creation and RSVPs (AMAs, virtual meetups, merch drops).
- Ticketing system (free or gated for holders).
- Exclusive perks for verified holders (channels, early access, reward multipliers).

#### **Rewards & Quests**
- Simple daily/weekly quests (e.g., “Post your first meme”, “RSVP to an event”).
- Earn **Badge NFTs** (Sui Objects) + token drips.

### 5. Nice-to-Have (Post-MVP or Stretch Goals)
- AI-powered lofi music generator or Yeti meme creator (community voting → mint winner).
- Embedded mini Academy / guided onboarding quests.
- Full real-time chat.
- Merch redemption preview/integration.
- Deeper charity donation flows (micro-donations with impact NFTs).

### 6. User Flows (Key Examples)
1. **New User Onboarding:** Connect wallet → Gasless profile creation → Welcome quest → First meme post.
2. **Daily Engagement:** Open app → Check Flurry Dashboard → Post meme or react → Earn reward.
3. **Event Participation:** Browse events → RSVP on-chain → Get reminder + badge.

### 7. Technical Requirements
- **Blockchain:** Sui Mainnet/Testnet (Move language).
- **Key Primitives:** Sui Objects (profiles, badges), Kiosk (NFT trading), Sponsored Transactions (Enoki or Gas Pool), SuiNS integration.
- **Frontend:** Next.js / React + TailwindCSS + Framer Motion (chill animations, mobile-first).
- **Storage:** Sui for metadata + Walrus for larger files (memes, music).
- **Wallet:** WalletConnect + zkLogin support.
- **Design System:** Colorful Yeti aesthetic (ice blues, purples, greens), lofi background option, flurry/ice particle effects, heavy use of existing Lofi mascot style.

### 8. Non-Functional Requirements
- Fully responsive (mobile priority).
- Fast & cheap (leverage Sui parallel execution).
- Secure (standard Move best practices, no admin keys post-launch if possible).
- Accessible & fun branding.

### 9. Roadmap
- **MVP (Hackathon):** Features in Section 4 + polished UI + demo video.
- **Phase 2 (Post-Hackathon):** AI tools, full chat, deeper Academy integration, mainnet launch with Lofi team.
- **Phase 3:** Merch bridge, advanced analytics, cross-platform (if mobile app).

### 10. Risks & Assumptions
- **Risks:** Time constraint for complex Move contracts; media storage costs; community adoption.
- **Mitigations:** Focus on core loop first; use existing Sui templates/libraries.
- **Assumptions:** Access to Lofi branding assets; possible collaboration with Lofi team (as encouraged); public Foundation data available.
